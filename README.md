# ✈️ Flight Price Analysis - EDA

## 📌 About
Exploratory Data Analysis on Indian flight prices
to find patterns and insights.

## 🔍 Key Questions Answered
- Which airline is cheapest?
- Does booking early save money?
- Which route is most expensive?
- Best value airline (price per hour)?

## 🛠️ Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## 📊 Key Insights
- AirAsia is the cheapest airline
- Booking early generally saves money
- More stops = higher price
